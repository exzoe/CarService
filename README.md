# Car Service
### Ссылка на видео - https://youtu.be/ZHowVTXSKUE (записывал с компа демку с ноута, сори за качество)
Backend-сервис для управления автомобилями и водителями, реализованный на Spring Boot с использованием базы данных PostgreSQL. Сервис предоставляет REST API для управления автомобильным парком.

## Описание
Сервис предоставляет REST API для:

управления водителями (создание, обновление, удаление)

управления автомобилями (добавление, редактирование, удаление)

управления типами автомобилей (седан, внедорожник и т.д.)

назначения водителей на автомобили

хранения информации о водителях (имя, возраст)

хранения информации об автомобилях (марка, модель, год выпуска, тип)

Сервис не содержит пользовательского интерфейса и предназначен для работы через HTTP-запросы.

## Стек технологий
Java 21

Spring Boot 3.2.0

Spring Web

Spring Data JPA

PostgreSQL 15

Gradle

Docker / Docker Compose

## Структура базы данных
drivers - таблица водителей

cars - таблица автомобилей

car_types - таблица типов автомобилей

## Основные эндпоинты
### Водители (Drivers)
GET /api/drivers - получить всех водителей

GET /api/drivers/{id} - получить водителя по ID

POST /api/drivers - создать нового водителя

### Автомобили (Cars)
GET /api/cars - получить все автомобили

GET /api/cars/{id} - получить автомобиль по ID

POST /api/cars - создать новый автомобиль

POST /api/cars/{carId}/driver/{driverId} - назначить водителя на автомобиль

### Типы автомобилей (Car Types)
GET /api/car-types - получить все типы автомобилей

GET /api/car-types/{id} - получить тип автомобиля по ID

POST /api/car-types - создать новый тип автомобиля
